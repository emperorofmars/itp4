# itp4
Turn Based Strategy Game based on [mgf](https://github.com/emperorofmars/mgf)

#### Install and run:
Compile with cmake:<br>
mkdir build<br>
cd build<br>
cmake ..<br>
make<br>
cd ../bin
ln -s ../res res<br>
ln -s ../libs libs<br>

run:<br>
./itp4
